About Electron Shell Browser 

electron-browser-shell is a minimal, tabbed web browser with support for (some) Chromium extension APIs.

It is built on Electron, and utilizes two npm packages to achieve its functions. They are:

  • electron-chrome-context-menu - a module that aims to painlessly recreate the Chromium native context menu, with (close to) feature parity with Chromium.

  • electron-chrome-extensions - a library that aims to recreate some missing or incomplete Chromium extension APIs in Electron.
Electron has partial support for a subset of these APIs, as per Here, but usually not enough to meaningfully utilize popular extensions on the Chrome Web Store.
This app can be used standalone to test the progress of Electron Chrome Extensions, as a minimal browser for fun, or as a boilerplate to create your own Electron based browser.
